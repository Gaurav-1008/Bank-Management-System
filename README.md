# Bank Management System

## Overview
A Java-based Bank Management System using JavaFX, MySQL, and email authentication.

## Features
✅ User authentication  
✅ Admin panel  
✅ Deposit & withdrawal  
✅ Email-based password recovery  
✅ Dark theme UI

## Setup
1. Import `bank_management.sql` into MySQL.
2. Update `DBConnection.java` with database credentials.
3. Run `Main.java` to start the application.
